◆作品について◆

◇使用言語 : C++
◇SDK      : DirectX9
◇開発期間 : 2022/11～2022/02(3か月)
◇ジャンル : 3Dアクションゲーム

◆動作確認済み環境◆
Windows10
（DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆操作方法◆
この
◇ゲームパッド
Lスティック...移動
Rスティック...カメラ移動
A.............ドアを開ける、鍵をとる、ハッキング、ボタンを押す
Y.............索敵
X.............プレイヤーとロボットの操作切り替え
B.............ステルスキル(敵に見つかってないときに後ろから)
R.............銃を構える
L.............銃を撃つ(銃を構えた状態のみ)

◇キーボード	
WASD..........移動
← →.........カメラ移動
E.............ドアを開ける、鍵をとる、ハッキング、ボタンを押す
Q.............索敵
V.............プレイヤーとロボットの操作切り替え
F.............ステルスキル(敵に見つかってないときに後ろから)
↑............銃を構える
SPACE.........銃を撃つ(銃を構えた状態のみ)
